# COBOL ATM Banking System

## Overview
The COBOL ATM Banking System is designed to simulate an ATM service for managing bank accounts. It includes functionalities like account creation, PIN authentication, handling transactions, and generating receipts for each transaction.

## Features
- **PIN Authentication**: Users can securely login to their accounts using a 4-digit PIN.
- **File Handling**: The system uses flat files to store user account data and transaction records. This ensures persistence and reliability in data management.
- **Transaction Management**: Users can deposit, withdraw, and check their balances seamlessly. The system maintains the integrity of transactions and ensures sufficient balance before processing withdrawals.
